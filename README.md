# AkPSI Network :handshake:
## Professional Networking Platform for Alpha Kappa Psi

This repository contains the architecture for the AkPSI Network, a tool designed to streamline professional connections and resource sharing within the University of Iowa chapter.

### :star: Key Features
* **Member Directory:** Dynamic access to professional profiles and networking data.
* **Resource Hub:** Centralized storage for professional development materials.

### :computer: Setup & Contribution
1. **Clone Repo:** `git clone https://github.com/hbsaine/AkPSI-Network.git`
2. **Install Dependencies:** `npm install`
3. **Local Development:** `npm start` to view the platform locally.
